API Overview

Welcome to Lumanu, a modern and fast text extraction utility designed specifically for AI and large language model applications. Most public websites are designed to look great to human users, filled with structural menus, advertising trackers, stylesheets, and scripting structures. When sent directly to LLMs, this introduces severe context inflation, raising your token usage costs and introducing unnecessary noise that degrades logic processing accuracy.

Lumanu runs high-performance extraction microservices right at the edge network (powered by Cloudflare Workers) to fetch webpages, clean up HTML layers, and provide neat, structurally-correct markdown output instantly.

Note: Lumanu's endpoint is completely public and open for general developer usage at https://lumanu.mcbcode.com.

Quickstart with cURL

To run your very first webpage text extraction, you can query our global extract endpoint directly from your terminal workspace using cURL.

bash
$ curl "https://lumanu.mcbcode.com/extract?url=https://example.com"

This outputs a JSON response containing metadata about the target webpage and the extracted clean text content block.

GET /extract

Our main interface for fetching and cleaning webpages. You communicate with this endpoint by appending URL variables via structured GET query parameters.

Parameter Type Requirement Description
url String Required The target webpage address to fetch (e.g., https://example.com). Make sure it includes the protocol prefix.
format String Optional Desired output formatting. Choices include: json, text, markdown, yaml. Defaults to json.

Response Format

When requesting data in standard JSON format, you can expect a flat, reliable JSON schema structured as follows:

json
{ "success": true, "url": "https://example.com", "metadata": { "title": "Example Domain" }, "text": "This domain is for use in illustrative examples in documents..." }

LLM Token Optimization

Webpages often contain thousands of lines of code. By extracting only raw text or markdown format, you remove massive CSS sheets and dynamic tracking script blocks. Here is how that impacts context windows:

Page Resource Raw HTML Tokens Lumanu Markdown Tokens Savings
Standard documentation page ~42,000 ~1,800 95.7% Reduction
Web log article ~12,000 ~950 92.1% Reduction

This severe compression allows you to feed deep website pages directly into LLMs without blowing past rate-limits or filling models with useless context garbage.

Self Hosting Lumanu

Because Lumanu is completely open-source, you can easily self-host the deployment within your own private infrastructure. Check the root level repository layout for more detailed deployment directions:

To deploy the utility onto a local Cloudflare Worker environment, navigate to your terminal instance and run:

bash
# Clone code repository $ git clone https://github.com/xyztoast/lumanu.git $ cd lumanu # Run wrangler deploy to push code directly to Cloudflare $ npx wrangler deploy
